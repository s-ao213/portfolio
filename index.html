オリジナルテトリス

Python

2023年、12月に1年間のPython学習の集大成として作品しました。既存のゲームであるテトリスをイメージして、下の画像にあるようなミニゲームを作成しました。左右の矢印キーでブロックの位置、上下の矢印キーでブロックの向きを変更することができます。またenterキーでブロックを投下することができ、横1列のマスが埋まると消えてスコアが100点が加算されます。2分間で1000点取れるとクリアとなっています。また画面の一番上までブロックがたまると強制的にGAME OVERになるようにしました。 ほかにも画面右上で次に出てくるブロックを確認でき、2回目以降は前回の記録も確認できるようになっているのでPythonの実行環境のある方は誰でもプレイしていただけます。ブロックが自動で落ちてくる機能は、まだ実装できていません。

カレンダーアプリ開発

Python PySide

2024年1月に12時間ほどかけてPySideでカレンダーアプリを開発しました。このアプリはユーザーが日付を選択し、予定を追加できる機能を持っています。アプリを起動している間、入力された予定はメモリに保存され、ユーザーは情報を上書きすることができます。さらに予定は家、学校、職場、プライベート、その他で分類・色分けをされます。登録された予定の日付のマスは、それぞれの色で表示されるため、ユーザーがより直感的に予定を把握することができます。

JavaScriptの学習支援サイト

HTML CSS JavaScript

2024年7月にHTML、CSS、JavaScriptを用いて初心者向けのJavaScript学習支援サイトを作成しました。制作時間は20時間程度です。このサイトはWeb上で動く簡単な迷路ゲームを作成しながら、JavaScriptの基礎が学べるようになっています。初心者の方でもプログラミングに興味を持てるようにと、迷路ゲームの作成をメインにしました。このサイトでは、プログラミングの基礎となる考え方の、変数とデータ型、関数の定義と使用、条件分岐とループなどを学べます。

Reactを用いたオリジナルTodoアプリ

React

2024年12月にReactを用いてオリジナルのタスク管理アプリを作成しました。製作時間は25時間程度です。このサイトではローカルにデータを保存し、ページを閉じても実際の端末上には、タスクのデータが残るようになっています。また、作成したタスクの進捗状況をパーセンテージで表示することでユーザーが直観的に進捗を理解できるようになっています。タスクの検索やカテゴリ分けなどのオリジナル機能を実装しました。詳しくはRepositoryのREADME.mdをご覧ください。

Reactを用いたオリジナルchatアプリ

React Next.js

2025年2月に作成したchatアプリです。製作期間は3か月程度で簡単かつセキュリティの高いWebチャットアプリとなっています。招待制のため、承認されたユーザー同士でのみチャットが可能です。メッセージは暗号化して保存され、30件を超えると古いmessageから自動削除されるため容量も軽いです。またアイコンや表示名のカスタマイズ機能も備えておりユーザー目線でのUI/UXにも力を入れました。詳しくはRepositoryのREADME.mdをご覧ください。
